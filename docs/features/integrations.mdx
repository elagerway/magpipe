## Overview

Integrations let your AI agents work with external services. Sync contacts, book appointments, and automate workflows.

## Available Integrations

### HubSpot

Sync contacts and companies with HubSpot CRM.

**Features:**
- Import contacts from HubSpot
- Create new contacts when unknown callers call
- Log calls and messages to contact timeline
- Sync company information

**Setup:**
1. Go to **Apps** in the main navigation
2. Find HubSpot and click **Connect**
3. Authorize Solo Mobile in HubSpot
4. Select which data to sync

### Google Calendar

Let agents check availability and book appointments.

**Features:**
- Check your calendar for availability
- Book appointments during calls
- Send calendar invites to callers
- Respect busy times and working hours

**Setup:**
1. Go to **Apps** in the main navigation
2. Find Google Calendar and click **Connect**
3. Sign in with your Google account
4. Grant calendar access

### MCP Servers (Advanced)

Connect to Model Context Protocol servers for custom tool access.

**Features:**
- Add custom tools for your agents
- Connect to internal APIs
- Extend agent capabilities

**Setup:**
1. Go to **Apps** in the main navigation
2. Click **Add MCP Server**
3. Enter the server URL
4. Configure available tools

<Note>
MCP servers require technical setup. See the [MCP documentation](https://modelcontextprotocol.io) for details.
</Note>

## Integration Settings

Each integration can be configured per-agent:

1. Open the agent you want to configure
2. Go to the **Integrations** tab
3. Enable/disable specific integrations
4. Configure integration-specific settings

## Data Privacy

- Solo Mobile only accesses data you explicitly authorize
- You can disconnect integrations at any time
- Disconnecting removes Solo Mobile's access to that service
