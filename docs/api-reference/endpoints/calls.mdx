## Initiate Call

Start an outbound call with an AI agent.

<ParamField body="to" type="string" required>
  Phone number to call (E.164 format)
</ParamField>

<ParamField body="from" type="string" required>
  Your Solo Mobile phone number to call from
</ParamField>

<ParamField body="agent_id" type="string" required>
  Agent UUID to handle the call
</ParamField>

```bash
curl -X POST "https://mtxbiyilvgwhbdptysex.supabase.co/functions/v1/initiate-bridged-call" \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "to": "+14155551234",
    "from": "+16045551234",
    "agent_id": "agent-uuid"
  }'
```

### Response

```json
{
  "success": true,
  "call_id": "call-uuid",
  "status": "initiated"
}
```

## Get Call

Retrieve call details including recording and transcript.

<ParamField path="id" type="string" required>
  Call UUID
</ParamField>

```bash
curl -X GET "https://mtxbiyilvgwhbdptysex.supabase.co/rest/v1/call_logs?id=eq.CALL_ID&select=*" \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN" \
  -H "apikey: YOUR_ANON_KEY"
```

### Response

```json
{
  "id": "call-uuid",
  "from_number": "+16045551234",
  "to_number": "+14155551234",
  "direction": "outbound",
  "status": "completed",
  "duration": 120,
  "recording_url": "https://...",
  "transcript": "Agent: Hello! Caller: Hi there...",
  "created_at": "2024-01-15T10:00:00Z"
}
```

## List Calls

Get all calls for the authenticated user.

<ParamField query="limit" type="integer">
  Number of results (default: 50)
</ParamField>

<ParamField query="offset" type="integer">
  Pagination offset
</ParamField>

```bash
curl -X GET "https://mtxbiyilvgwhbdptysex.supabase.co/rest/v1/call_logs?select=*&order=created_at.desc&limit=50" \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN" \
  -H "apikey: YOUR_ANON_KEY"
```

### Response

```json
[
  {
    "id": "call-uuid",
    "from_number": "+16045551234",
    "to_number": "+14155551234",
    "direction": "inbound",
    "status": "completed",
    "duration": 45,
    "created_at": "2024-01-15T10:00:00Z"
  }
]
```

## Call Statuses

| Status | Description |
|--------|-------------|
| `initiated` | Call is being placed |
| `ringing` | Phone is ringing |
| `in-progress` | Call is active |
| `completed` | Call ended normally |
| `busy` | Recipient was busy |
| `no-answer` | No answer |
| `failed` | Call failed to connect |
