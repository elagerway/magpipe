## List Agents

## Create Agent

Create a new AI agent.

<ParamField body="name" type="string" required>
  Agent name
</ParamField>

<ParamField body="greeting" type="string">
  Greeting message for calls
</ParamField>

<ParamField body="system_prompt" type="string" required>
  System prompt defining agent behavior
</ParamField>

<ParamField body="voice_id" type="string">
  Voice ID for phone calls
</ParamField>

```bash
curl -X POST "https://mtxbiyilvgwhbdptysex.supabase.co/rest/v1/agent_configs" \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN" \
  -H "apikey: YOUR_ANON_KEY" \
  -H "Content-Type: application/json" \
  -H "Prefer: return=representation" \
  -d '{
    "name": "My Agent",
    "greeting": "Hello! How can I help you?",
    "system_prompt": "You are a helpful assistant for Acme Corp...",
    "voice_id": "alloy"
  }'
```

### Response

```json
{
  "id": "new-uuid",
  "name": "My Agent",
  "greeting": "Hello! How can I help you?",
  "system_prompt": "You are a helpful assistant for Acme Corp...",
  "voice_id": "alloy",
  "created_at": "2024-01-15T10:00:00Z"
}
```
