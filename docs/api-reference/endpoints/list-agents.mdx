---
title: "List Agents"
api: "POST /list-agents"
description: "List all AI agents in your account"
---

Retrieve all AI agents configured in your account.

## Request Body

<ParamField body="limit" type="integer" default={50}>
  Maximum agents to return. Max: 100.
</ParamField>

<ParamField body="offset" type="integer" default={0}>
  Number of records to skip for pagination.
</ParamField>

<ParamField body="is_active" type="boolean">
  Filter by active status.
</ParamField>

<ParamField body="agent_type" type="string">
  Filter by type: `inbound`, `outbound`, or `both`.
</ParamField>

## Response

<ResponseField name="agents" type="array">
  Array of agent objects.
</ResponseField>

<ResponseField name="total" type="integer">
  Total number of agents in account.
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST https://api.magpipe.ai/functions/v1/list-agents \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"is_active": true}'
```

```javascript Node.js
const response = await fetch(
  'https://api.magpipe.ai/functions/v1/list-agents',
  {
    method: 'POST',
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY',
      'Content-Type': 'application/json',
    },
    body: JSON.stringify({ is_active: true }),
  }
);

const { agents, total } = await response.json();
console.log(`You have ${total} agents`);
```

```python Python
import requests

response = requests.post(
    'https://api.magpipe.ai/functions/v1/list-agents',
    headers={
        'Authorization': 'Bearer YOUR_API_KEY',
        'Content-Type': 'application/json',
    },
    json={'is_active': True}
)

data = response.json()
print(f"You have {data['total']} agents")
```
