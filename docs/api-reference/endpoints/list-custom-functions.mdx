---
title: "List Custom Functions"
api: "GET /custom-functions"
description: "List all custom functions for an agent"
---

Retrieve all custom functions configured for a specific agent.

## Query Parameters

<ParamField query="agent_id" type="string" required>
  UUID of the agent to list functions for
</ParamField>

## Response

Returns an array of custom function objects.

<ResponseField name="id" type="string">
  UUID of the function
</ResponseField>

<ResponseField name="agent_id" type="string">
  UUID of the agent
</ResponseField>

<ResponseField name="name" type="string">
  Function name
</ResponseField>

<ResponseField name="description" type="string">
  Function description
</ResponseField>

<ResponseField name="http_method" type="string">
  HTTP method (GET, POST, etc.)
</ResponseField>

<ResponseField name="endpoint_url" type="string">
  Webhook URL
</ResponseField>

<ResponseField name="is_active" type="boolean">
  Whether the function is active
</ResponseField>

<ResponseField name="created_at" type="string">
  ISO 8601 timestamp
</ResponseField>

<RequestExample>
```bash cURL
curl "https://api.magpipe.ai/functions/v1/custom-functions?agent_id=123e4567-e89b-12d3-a456-426614174000" \
  -H "Authorization: Bearer YOUR_API_KEY"
```
</RequestExample>
